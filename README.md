# list-crud
List and items API and Client

##Prerequisites
To Install and run the project you need to have installed NodeJS
``` 
https://nodejs.org/en
```

and MongoDB
``` 
https://www.mongodb.com/download-center?jmp=nav#community
```

MongoDB should be setup as
``` 
    "host": "localhost",
    "port": 27017,
    "password": "",
    "user": "",
```

##Run API Server
Open a terminal or cmd and type
```
$ cd list-crud
$ npm start
```

The server will run on:
```
http://localhost:3000
```

The api will run on:
```
http://localhost:3000/api
```

##Run Client
Open another terminal or cmd and type
```
$ cd list-crud
$ cd client
$ npm start
```

The client will run on:
```
http://localhost:4200
```
